# BusZ Mobile App - Yêu cầu chức năng

## 1. Mục tiêu hệ thống
BusZ là ứng dụng đặt vé xe khách liên tỉnh, hỗ trợ người dùng tìm chuyến, xem chi tiết nhà xe, chọn ngày, chọn ghế, nhập thông tin hành khách, thanh toán và quản lý vé điện tử.

## 2. Đối tượng sử dụng
- Khách hàng
- Nhà xe
- Quản trị viên hệ thống

## 3. Luồng chính của ứng dụng
Search tuyến xe
→ Chọn ngày đi / ngày về
→ Xem danh sách chuyến
→ Xem chi tiết nhà xe
→ Chọn chuyến đi
→ Chọn chuyến về nếu khứ hồi
→ Chọn số ghế
→ Nhập thông tin liên hệ
→ Nhập thông tin hành khách
→ Chọn ghế
→ Thanh toán
→ Nhận vé điện tử
