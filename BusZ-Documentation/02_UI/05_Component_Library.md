# 1. Purpose

Component Library là thư viện chứa toàn bộ UI Component được sử dụng trong BusZ.

Mục tiêu:

- Tái sử dụng Component.
- Đồng bộ giao diện.
- Giảm thời gian phát triển.
- Giảm lỗi UI.
- Đồng nhất giữa Flutter và Figma.

# 31. Summary

Component Library là tài liệu định nghĩa toàn bộ thành phần giao diện của BusZ.

Tất cả Widget Flutter, Component React và Figma Component phải được xây dựng dựa trên tài liệu này nhằm đảm bảo tính thống nhất, khả năng tái sử dụng và dễ bảo trì.
