# 1. Purpose

Typography định nghĩa toàn bộ hệ thống chữ sử dụng trong BusZ.

Mục tiêu:

- Đồng nhất giao diện.
- Dễ đọc.
- Dễ bảo trì.
- Dễ mở rộng.
- Hỗ trợ nhiều ngôn ngữ.

Typography phải được sử dụng thống nhất trên:

- Flutter Mobile
- Admin Website
- Figma
- Documentation

---

# 2. Design Principles

Typography phải:

- Dễ đọc
- Rõ ràng
- Có thứ bậc
- Khoảng cách hợp lý
- Không lạm dụng Font Weight
